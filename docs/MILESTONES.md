# Completion Rules

A milestone is complete only if:

* Feature works
* Security verified
* Mobile responsive
* Documentation updated
* No TypeScript errors
* No lint errors
* No console errors
* Existing functionality preserved

Do not begin the next milestone until the current one is complete.
